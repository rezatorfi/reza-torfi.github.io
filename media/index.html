Computers have been a part of my life from a very young age and I have been intrigued to not only see, but also to experience first hand the progress of computers and how they have become ubiquitous in almost all areas of our lives.

iTunes UK Charts

For this project, I built a website that shows the most up-to-date chart information from iTunes. You can select from various different genres as well as albums or songs. The chart information is downloaded from the iTunes API using AJAX and is then appended to the screen with JavaScript. This site is also fully responsive to different screen sizes and ratios.

Project #2

Seeking to continue my learning, I decided to take on the challenge of building a game. To do this I used a JavaScript library called Phaser. This not only allowed me to continue improving my JavaScript skills, but also to take the time to learn the ins and outs of a fully featured JavaScript library. I also wrote the back end code for this app and I used Node.js and Express for this, and hosted it on Heroku. This allowed me to apply a persistant scoreboard for the game which would be the same for everyone, regardless of which computer they accessed the site on.

Project #4

Seeking to improve on my React skills, I wanted to take on a project that would really test my skills. To do this, I decided to build my own version of mathematician John Horton Conway's Game Of life. Designed in 1970, the Game Of Life simulates the life cycles of cells in a two-dimensional grid. This presented various unique challenges, such as how to implement the rules that dictate cell's life cycles, and creating a grid that would wrap both horizontally and vertically.
